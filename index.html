About

Elicit is a browser-based tool for picture-based elicitation of spoken language samples from participants using pictorial stimuli.

Upload images, record responses, and download the audio as .wav files for analysis.

③ How to use
1
Enter session details using button 1 — participant ID, age, language, stimulus set, and so on. A Session ID is auto-generated but can be edited.
2
Upload story images using button 2. Navigate with the → arrow. Hover a thumbnail and press × to remove an image.
3
Press Record to begin. Press Stop when done. A tick mark on the thumbnail confirms a saved recording.
4
To re-record the current image, press Redo. Press Play in the Notes panel to review any recording before downloading.
5
Press Download to export .wav files, a _session.csv with full metadata and timestamps, and a .cha CHAT header file ready for CLAN.
6
Use ↺ Reset in the session bar to clear all data and start a new participant. You will be prompted to download first if recordings exist.
Recording continues across images unless manually stopped. Stop and re-record per image to save separate files.
